# CoverAI - краткий бизнес-план

## Суть

CoverAI помогает кандидату быстро получить персональное сопроводительное письмо
под конкретную вакансию hh.ru: пользователь один раз сохраняет резюме, отправляет
ссылку на вакансию и получает готовый текст через Telegram-бота или API.

## УТП

| Для пользователя | Что дает CoverAI |
| --- | --- |
| Нет времени писать письма под каждую вакансию | Письмо генерируется за один короткий сценарий: резюме -> ссылка hh.ru -> текст |
| Шаблонные письма плохо работают | Письмо строится по резюме, вакансии, работодателю и выбранному тону |
| Не хочется заводить кабинет | Основной сценарий работает прямо в Telegram без email/password |
| Нужен контроль расходов | Кредиты списываются только за успешную генерацию; ошибки hh.ru/LLM не оплачиваются |
| Нужна история и варианты подачи | Standard/Pro открывают историю писем и выбор тона: formal, confident, concise |

## Для кого

- Соискатели, которые откликаются на много вакансий.
- Junior/middle-специалисты, которым сложно быстро формулировать сильный отклик.
- Карьерные консультанты и помощники, готовящие письма для нескольких клиентов.
- Будущие B2B/API-клиенты: карьерные сервисы, боты, HR-инструменты.

## Финмодель

| Источник денег | Механика |
| --- | --- |
| Пополнение кредитов | 1 успешная генерация = `PREDICTION_COST_CREDITS`, по умолчанию 1 кредит |
| Подписка Standard | 399 RUB / 30 дней: 300 генераций за календарный месяц, 3 тона, история за 30 дней |
| Подписка Pro | 999 RUB / 30 дней: безлимит генераций, 3 тона, полная история |
| Промокоды | Welcome credits, фиксированные кредиты или скидка на следующее пополнение |

Подписка продает возможности и лимиты, а кредиты отдельно оплачивают успешные
LLM-генерации. Это снижает риск: пользователь не платит за технические ошибки, а
сервис сохраняет переменную выручку с активных пользователей.

## Юнит-экономика

| Показатель | Базовое допущение из документации |
| --- | --- |
| Цена кредита | `CREDIT_PRICE_RUB = 1` RUB |
| Стоимость генерации для пользователя | 1 кредит за успешное письмо |
| Переменная LLM-себестоимость | около `$0.01` за письмо |
| Валовая маржа | цена письма минус LLM, платежные и инфраструктурные расходы |

Ключевая формула:

```text
Маржа с письма = цена кредита * кредитов за генерацию - LLM cost - payment fee - infra cost
```

Главная цель MVP - проверить конверсию из бесплатного Telegram-сценария в
пополнение кредитов и Standard/Pro, а также фактическую частоту повторных
генераций.
